# Pass — layout-cleanup (deck)

Highest-leverage pass on slides. Cleans up layout, type, alignment, and density so the deck looks intentional. Cleanup, **not** redesign — never touch the narrative.

## Apply when

"cleaner slides", "fix the layout", "too much text", "tighten", "more professional", "make it look designed". Also good as a default refinement on any premium deck.

## Audit → fix checklist

- **One idea per slide** — split overloaded slides; if a slide has two points, make two slides.
- **Kill bullet walls** — convert bullets to a claim + one supporting visual, or a short phrase set. No paragraph dumps.
- **Type hierarchy** — clear title / body / caption steps; ≤ 2 families; consistent sizes *across* slides (same title size everywhere).
- **Grid + safe-area margins** — consistent gutters; nothing jammed to the edge; aligned to a shared grid.
- **Alignment** — shared alignment across slides; no drifting text boxes.
- **Contrast + legibility** — readable from the back of a room: min body size, strong contrast (light text on dark or vice versa, AA+).
- **Whitespace** — one focal point per slide; let slides breathe.
- **Consistency** — same footer/page-number/logo treatment on every slide.

## Modes

- audit-only → list findings, change nothing
- conservative (default) → fix clear problems, preserve the rest
- standard → conservative + normalize the full type/spacing system deck-wide

## Hard boundary — no narrative change

Fix **visual treatment** only. Never reorder slides, cut content, or rewrite the argument — that is the strategist's domain. If a slide is overloaded, split it; don't delete the point.

## Output

Per-slide findings (problem → fix) and the normalized layout/type system.
