# Pass - reference-enrichment (deck)

Anti-generic slide taste and art direction. Runs first - before or alongside
the strategist writing the deck - so the whole deck starts from a strong
aesthetic position. Inspiration, **not** imitation.

## Apply when
"less generic", "premium", "stronger design", "better taste", "more modern",
"high-end deck", or premium fidelity is implied.

## Do
- Set a **deck-wide visual system**: one display + one text typeface, a
  restrained palette (one accent used with discipline), a consistent grid.
- **One big idea per slide.** Each slide makes a single point.
- Editorial, intentional composition - asymmetry where it helps, generous
  whitespace, a clear focal point per slide.
- Produce a 1-2 sentence **art-direction note** (mood, type pairing, palette)
  the rest of the deck follows.
- For brand decks, ground direction in the client's brand (or a brandkit-style
  system), not a generic template.

## Don't (anti-generic checklist)
- no clipart, no stock-photo filler, no default-template look
- no bullet-soup (that's layout-cleanup's job to kill, but don't introduce it)
- no rainbow of accent colors; no 4+ typefaces
- no gradient-on-everything

## Hard boundary - inspiration, not cloning
Reference informs **direction** (mood, rhythm, restraint), never a 1:1 copy of
a named company's deck, assets, or slide order. If asked to "clone X's deck",
decline the clone and offer an original direction inspired by what makes X work.

## Output
Art-direction note + enriched deck brief, handed to the strategist.
